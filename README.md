## рҹ’Ў The Problem

> **70% of disaster response delays stem from coordination gaps.**  
> When a crisis strikes, fragmented tools, slow dispatching, and zero predictive intelligence cost precious time вҖ” and lives.

**ReliefLink AI** solves this by centralizing disaster intelligence into a single, AI-powered Command Center that doesn't just *react* вҖ” it **predicts**.

## рҹҡҰ Quick Start

```bash
# 1пёҸвғЈ  Clone
git clone https://github.com/vaishnavi-ctrl-jpg/RELIEFLINK-AI-
cd RELIEFLINK-AI-

# 2пёҸвғЈ  Install
npm install

# 3пёҸвғЈ  Configure (.env.local)
GOOGLE_CLOUD_PROJECT=your-project-id
GOOGLE_CLOUD_LOCATION=us-central1
GOOGLE_APPLICATION_CREDENTIALS=./service-account-key.json

# 4пёҸвғЈ  Launch
npm run dev
```

> рҹ’Ў **Need a Service Account key?**  
> [IAM Console](https://console.cloud.google.com/iam-admin/serviceaccounts) вҶ’ Create Account вҶ’ **Vertex AI User** role вҶ’ Keys вҶ’ JSON
